# Mica Minecraft Launcher -- User Guide

The **Mica Minecraft Launcher** is a cross-platform Minecraft Forge modpack launcher that makes it easy to install, update, and play modpacks with a single click. It supports Windows, macOS, and Linux, handles Java runtime management automatically, and includes built-in malware scanning and file integrity verification.

This guide covers every feature of the launcher in detail, from first-time setup to advanced configuration.
