Main Screen

The main screen is your home base after signing in. From here you can browse installed modpacks, launch the game, and navigate to settings or modpack management.

Modpack List

The left panel displays all installed modpacks. Each entry shows the modpack name and the last played time beneath it. Select a modpack to view its details in the center area.

When a modpack has a newer version available, a green UPDATE badge appears next to its name. The update is applied automatically the next time you click Play.

Tip

Right-click any modpack in the list for a context menu with extra options:

  • Open Modpack Folder — browse the local files on disk
  • Open Mods Folder — jump straight to the mods directory
  • Create Desktop Shortcut — launch this modpack directly from your desktop
  • Play Statistics — view total play time and session history

Center Panel

Selecting a modpack displays its description and a background image in the center area. Two primary buttons appear at the bottom:

  • Play — downloads any missing or updated files, then launches the game.
  • Website — opens the modpack's homepage in your browser (if one is defined).

Navigation

  • Settings — opens launcher settings (memory, theme, JVM flags, and more).
  • Edit Mod Packs — opens the modpack manager to add or remove modpacks.
Note

If you have no modpacks installed, the center panel will prompt you to add one via Edit Mod Packs.
