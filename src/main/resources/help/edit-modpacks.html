Edit Mod Packs

The modpack management screen lets you add, remove, and organize the modpacks available in your launcher.

Adding Modpacks

Add by URL

If you have a direct URL to a modpack's installable.json manifest, paste it into the URL field and click Add. The launcher will download the manifest, validate it, and add the modpack to your list.

Add from List

Click Add from List to browse a curated list of pre-approved modpacks. Select one or more packs and confirm to add them.

Tip

Modpacks added by URL will receive automatic updates whenever the hosted manifest changes, just like packs added from the list.

Removing Modpacks

Select a modpack and click Remove Selected to uninstall it from the launcher. This removes the modpack entry and its downloaded files from disk.

Generate Hosting Manifest

The Generate Hosting Manifest button creates an installable.json file from an existing modpack on disk. This manifest is the standard format the launcher uses to install and update modpacks. You can host the resulting file on any web server or file hosting service so that other users can add your modpack by URL.

The manifest includes:

  • Modpack metadata (name, version, description, Minecraft and Forge versions)
  • File lists with download URLs and SHA-1 hashes for integrity verification
  • Optional background image and website URLs

Modpack Editor

For creating or editing modpack JSON definitions from scratch, use the Modpack Editor — a full-featured editor with validation, Modrinth search, and diff view.

Note

After adding or removing modpacks, return to the Main Screen to see the updated list.
