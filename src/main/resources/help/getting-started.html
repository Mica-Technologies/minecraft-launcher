Getting Started

Welcome to the Mica Minecraft Launcher — a cross-platform Minecraft Forge modpack launcher that makes it easy to install, update, and play modpacks with a single click.

Quick Start

  1. Sign in with your Microsoft account. The launcher uses official Microsoft authentication to verify your Minecraft ownership. See Login for details.
  2. Select a modpack from the list on the left side of the Main Screen. If no modpacks are installed yet, use Browse to add one.
  3. Click Play and the launcher will automatically download all required files, install the correct Forge version, verify file integrity, and launch the game.

What the Launcher Handles for You

  • Downloading and updating modpack files (mods, configs, resources)
  • Installing the correct Forge version for each modpack
  • Managing Java runtimes — no manual JDK installation needed
  • Verifying file integrity with SHA-1 hash checks
  • Scanning downloaded files for known malware
Tip

Look for the ? button on every screen — it opens context-sensitive help for the screen you are currently viewing.
