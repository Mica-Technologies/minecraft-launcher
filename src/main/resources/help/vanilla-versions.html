Vanilla Versions

In addition to Forge modpacks, the launcher can install and play vanilla (unmodded) Minecraft versions directly.

Browsing Versions

The vanilla versions screen displays a list of available Minecraft releases and snapshots. Each entry shows the version number and release date. Select a version to see its details.

Installing and Launching

  1. Select a Minecraft version from the list.
  2. Click Install to download the game client, libraries, and assets for that version.
  3. Once installed, click Play to launch. The launcher handles all file verification and runtime setup automatically.
Note

Vanilla versions use the same memory and JVM settings configured in Settings. If a version runs poorly, try adjusting your RAM allocation or JVM preset.

Runtime Management

Different Minecraft versions may require different Java runtimes. The launcher automatically downloads and manages the correct runtime for each version — for example, Minecraft 1.17+ requires Java 17 or later, while older versions use Java 8.

You do not need to install Java manually. See Runtime Management for more details on how runtimes are handled.

Tip

Vanilla versions are a good way to test whether your system meets the basic requirements before installing larger modpacks.
