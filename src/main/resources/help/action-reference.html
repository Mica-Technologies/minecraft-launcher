Action Reference

Every modpack action in the launcher can be reached from at least one canonical surface, with a context-menu duplicate for fast access when you're already looking at a card. This page documents where each action lives so you don't have to hunt.

Canonical Surfaces

SurfaceWhat lives here
Main menu hero cards Primary Play + Website buttons per pack. Click-through opens the detail modal.
Modpack Detail Modal The richest per-pack surface: Quick Actions tiles (Open Folder / Mods / Config / Screenshots / Resource Packs / Shaders / Desktop Shortcut / Copy Invite Link / Export as ZIP), Stats, Update Log, Worlds, Screenshots grid, Shader / Resource Packs lists, Crash History, Advanced section (verify), and Play / Website action buttons.
Browse screen cards Install / Uninstall / Edit depending on the entry's kind (modpack / vanilla / loader version).
Right-click context menu Fast access to the common Quick Actions from any pack card without opening the detail modal. Shares all the same handlers as the modal.
Settings Launcher-wide preferences: account, game / runtime, appearance, advanced, network, security, system, Discord, RGB, About.

Tip

Almost every per-pack action has a right-click context menu shortcut on the pack's card. If you find yourself repeatedly clicking through the detail modal just to reach the same folder, try right-clicking the card on the main menu — the same items live one click away.
