Keyboard Shortcuts

The launcher exposes a small set of keyboard shortcuts for the actions you'll do most often. On macOS the modifier is ⌘ Command; on Windows and Linux it's Ctrl. The shortcuts below show the modifier as ⌘/Ctrl to cover both.

Global Navigation

Available on every screen except Login, Launch Progress, and the in-game console window — those screens deliberately don't take navigation shortcuts so you don't accidentally walk away mid-flow.

ShortcutAction
⌘/Ctrl + ,Open Settings (universal convention)
⌘/Ctrl + LOpen the Browse screen (install & manage)
⌘/Ctrl + EOpen the Modpack Editor
⌘/Ctrl + Shift + MReturn to the Main menu (Home)
F1 / ⌘/Ctrl + ?Open the Help window on the current screen's topic
Why Cmd+Shift+M for Home?

Plain ⌘ + M is the macOS "Minimize Window" system shortcut — the launcher avoids overriding it. ⌘/Ctrl + Shift + M sidesteps the conflict on every platform.

Tip

The help button (?) in the top-right of every screen is the click-friendly way to surface this page or any other help topic. Hover its icon and you'll see the same tooltip the screen reader announces.
