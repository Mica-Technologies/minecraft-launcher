Keyboard Shortcuts

The launcher exposes a small set of keyboard shortcuts for the actions you'll do most often. On macOS the modifier is ⌘ Command; on Windows and Linux it's Ctrl. The shortcuts below show the modifier as ⌘/Ctrl to cover both.

Global (System Menu Bar)

Available everywhere as long as the launcher window has focus. They live on the system menu bar (top of the screen on macOS; top of the window on Windows / Linux).

ShortcutAction
⌘/Ctrl + LOpen the Browse screen (install & manage)
⌘/Ctrl + RRefresh the modpack list (re-fetch available manifests)
⌘/Ctrl + ?Open the Help window on this screen's topic

Main Menu

ShortcutAction
EnterPlay the last-selected modpack
F5Refresh available modpacks (same as the navbar refresh icon)

Modpack Detail Modal

ShortcutAction
EscClose the modal
Tip

The help button (?) in the top-right of every screen is the fastest way to surface the help topic for the surface you're on. The system menu bar's "Mica Minecraft Help" item always opens the Getting Started page.

Not Yet Bound

The following actions are commonly bound to shortcuts in other launchers but are not yet shortcut-bound here. They can still be accessed via the toolbar / menu bar / right-click context menus:

  • Settings (no shortcut — use the navbar Settings button)
  • Runtime Management (Game menu in the system menu bar)
  • Modpack Editor (Browse screen → bottom-bar Modpack Editor button)
  • Game Console window (auto-opens when you launch a pack with the console toggle on)
