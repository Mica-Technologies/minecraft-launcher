Runtime Management

The Mica Minecraft Launcher automatically manages Java runtimes so you never need to install or configure Java yourself.

Why Runtimes Matter

Minecraft and Forge require specific Java versions to run correctly. Different modpacks and Minecraft versions may need different Java runtimes:

  • Minecraft 1.16 and earlier — Java 8
  • Minecraft 1.17 - 1.20 — Java 17
  • Minecraft 1.21+ — Java 21 or later

The launcher detects which runtime each modpack or vanilla version requires and downloads it automatically on first launch.

Automatic Downloads

When you launch a modpack or vanilla version for the first time, the launcher checks whether the required runtime is already installed. If not, it downloads the correct version from Mojang's or Liberica's distribution servers. This is a one-time download per runtime version.

Note

Runtime downloads are typically 40 - 80 MB and are stored in the launcher's data directory, separate from any Java installation on your system.

Viewing Installed Runtimes

Open Settings and click Runtime Management to see a list of all installed Java runtimes, including their version, path, and which modpacks use them.

Deleting Unused Runtimes

If you have removed modpacks that used a particular Java version, you can delete the corresponding runtime to free disk space. Select a runtime in the management screen and click Delete.

Warning

Deleting a runtime that is still needed by an installed modpack will cause the launcher to re-download it on the next launch of that modpack.

Troubleshooting Runtimes

If a modpack fails to launch with Java-related errors, try the following:

  1. Open Runtime Management and delete the runtime used by the affected modpack.
  2. Launch the modpack again — the launcher will re-download a fresh copy.
  3. If the issue persists, see Troubleshooting for additional steps.
