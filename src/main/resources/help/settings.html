Settings

The settings screen lets you customize memory allocation, appearance, JVM behavior, network configuration, and more. Changes are saved automatically.

Game Memory

Minimum RAMThe initial heap size allocated to the game JVM. Typically leave at the default (512 MB) unless a modpack specifies otherwise.
Maximum RAMThe upper memory limit for the game. Most modpacks perform best with 4 - 8 GB. Allocating more than your system can spare may cause instability.
Tip

A good rule of thumb: set maximum RAM to no more than half your total system memory. For example, on a 16 GB machine, 6 - 8 GB is a reasonable upper limit.

Advanced Options

Debug ModeEnables verbose logging output in the launcher console for diagnosing issues.
Window ResizingAllows the launcher window to be freely resized beyond its default dimensions.
Discord IntegrationShows your current modpack as a Discord Rich Presence status.
Enhanced LoggingWrites detailed log files to disk for post-session analysis.
In-Game ConsoleOpens a Game Console window alongside Minecraft to view real-time output.

JVM Presets

Pre-configured sets of JVM flags appended to the game launch command:

Performance (Aikar's Flags)Recommended. Optimized garbage collection flags widely used by the Minecraft community. Best overall performance for most modpacks.
Low MemoryConservative flags for systems with limited RAM. Reduces GC overhead at the cost of some throughput.
DebugEnables JVM debug output including GC logs. Useful for diagnosing crashes or performance problems.
NoneNo extra flags — uses the JVM's defaults. Use this only if you know what you are doing or a modpack requires it.

Network / Proxy

Configure a proxy if your network requires one (common on corporate or school networks).

HTTP ProxyStandard web proxy. Enter the host and port provided by your network administrator.
SOCKS ProxyLower-level proxy protocol. Use this if your network specifically requires SOCKS.
Note

Most users can leave proxy settings blank. Only configure these if downloads or authentication fail on a restricted network.

Malware Scanning

The launcher includes a built-in scanner that checks downloaded JAR files for signatures of fractureiser — a Minecraft-specific malware that spread through compromised mods in 2023. The scan runs automatically before each game launch.

Reset and Runtime Options

Reset LauncherRestores all settings to defaults and clears cached data. This cannot be undone.
Runtime ManagementOpens the Runtime Management screen to view, verify, or remove installed Java runtimes.
Warning

Resetting the launcher will remove all settings, cached tokens, and modpack state. Your modpack files on disk are preserved, but you will need to re-add them.

Import / Export Settings

Use Export to save your current configuration to a file, and Import to load a previously saved configuration. This is useful for sharing settings between machines or restoring a known-good configuration after a reset.
