Browse

The Browse screen is your install & manage surface. From here you can install new modpacks from the public manifest, install pure vanilla Minecraft versions, install empty Forge / NeoForge / Fabric starter packs, edit packs you already have, and uninstall things you don't want any more.

Cards & the Filter Bar

Every entry shows up as a hero card with a logo, badge, name, chips (loader / Minecraft version), a status line, and action buttons. The filter bar across the top is how you narrow the view.

  • Search — free-text match against the card titles. Updates as you type (with a short debounce so it doesn't thrash the layout).
  • Type — All, Modpacks, or one of the version catalogues: Vanilla (Releases / Snapshots / Old Beta / Old Alpha) and Forge / NeoForge / Fabric versions. The loader-version filters are intentionally not included in "All" — the combined list would be thousands of entries and isn't what a "show me everything" search wants.
  • Status — All, Installed, or Available. "Available" shows things you haven't installed yet; "Installed" shows what's already on disk.
  • Sort — Default, Name (A-Z / Z-A), Release Date, Last Played, Most Played, Recently Updated. Sorts that depend on play / update history (Last Played, Most Played, Recently Updated) sink non-modpack entries to the bottom rather than mixing them in confusingly.

Installed Modpacks

Cards for packs already on disk show two buttons:

  • Uninstall — prompts whether to also delete the install folder. "Uninstall (Keep Files)" keeps your worlds, configs, and mods even though the launcher no longer knows about the pack — useful if you plan to reinstall the same pack later.
  • Edit — opens the Modpack Editor on the pack's local manifest. Changes are written to disk and apply on the next launch.
Tip

Right-click an installed modpack card for the same context menu the main menu offers: open the pack folder, open the mods folder, create a desktop shortcut, view play stats, and uninstall.

Available Modpacks

Cards from the public manifest you don't yet have show an Install button. Clicking it downloads the manifest, installs the modpack, and bumps the card into the Installed group on the next rebuild. You can keep using filters while an install is in flight — the in-progress card stays pinned at the top of the grid until the install finishes.

Vanilla Versions

Mojang's full release / snapshot / beta / alpha catalogue. Each card installs that exact Minecraft version as a synthetic empty modpack you can play directly from the main menu. Old betas and alphas are runnable too — the launcher pulls the appropriate Java runtime for each version automatically.

Forge / NeoForge / Fabric Versions

The loader-version filters surface every release advertised by each project's version service: Forge promotions, NeoForge maven metadata, and the Fabric meta API. Clicking Install on one of these writes a synthetic "empty modpack" manifest under your config folder and registers it with the launcher — so the result behaves like any other installed modpack:

  • shows up in the main menu's hero cards;
  • is editable in the Modpack Editor (so you can add mods on top to build your own pack);
  • can be uninstalled like any other pack;
  • shares the same auto-update plumbing as a normal modpack.
Note

The cards show each project's official logo when the launcher has it cached locally. The first visit to a loader filter renders the brand-coloured placeholder while the real logos download in the background — once cached, subsequent visits are instant.

Add by URL

The bottom-bar Add from URL button takes a direct URL to a hosted installable.json manifest and installs it the same way the available-pack cards do. Use this for modpacks distributed outside the public manifest — paste the link, click the button, and the manifest's metadata is downloaded and validated before install begins.

Modpack Editor

The bottom-bar Modpack Editor button opens the Modpack Editor with a blank manifest so you can build a new pack from scratch — set the loader (Forge / NeoForge / Fabric / vanilla), Minecraft version, mods list, etc. The Editor can also be opened on an existing pack via the Edit button on its card.

Generate Hosting Manifest

The bottom-bar Generate Hosting Manifest button creates an installable.json for the modpack currently in the editor, ready to host on any web server or file-sharing service. Anyone with the resulting URL can install your pack via Add from URL.

Pagination

Long lists are paginated. Use the Prev / Next buttons in the pagination bar (above the cards) to move between pages, and the Per page dropdown to set the page size — 24, 48, 72, or 96. The default of 48 fits the typical installed library on one page; bumping up to 96 keeps the loader-version filters scrollable in a single page.
