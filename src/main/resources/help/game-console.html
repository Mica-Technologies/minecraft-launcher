Game Console

The game console displays real-time output from the Minecraft process, including log messages, warnings, errors, and crash reports.

Enabling the Console

The console window appears alongside the game when In-Game Console is enabled in Settings under Advanced Options. When disabled, the game runs without a visible console.

Console Features

  • Real-time output — game log lines appear as they are produced.
  • Kill Game — forcefully terminates the Minecraft process if the game becomes unresponsive. Use this as a last resort.
  • Copy — copies the console contents to your clipboard for sharing or analysis.

Crash Reports

When the game crashes, the console automatically detects the crash and highlights the relevant crash report. You can:

  • View the full crash report directly in the console.
  • Use the Copy button to copy the crash report text for sharing with modpack authors or support channels.
Note

The console display is truncated to keep memory usage reasonable during long play sessions. For the complete, untruncated log, check the log files on disk.

Log File Location

Full game logs are written to the modpack's logs/ directory. You can access this folder quickly by right-clicking the modpack on the Main Screen and selecting Open Modpack Folder.

Tip

If you are reporting a crash, include both the crash report and the latest log file — the log often contains context that the crash report alone does not.
